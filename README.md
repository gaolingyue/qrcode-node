扫码登录Demo
====
  ## 模仿微信公众号的登陆流程 
  ### 输入名字 >> 生产二维码 >> 扫码  >>成功登录

 ### [强烈感谢 jeromeetienne 这个二维码生产库](https://github.com/jeromeetienne/jquery-qrcode)
